
This project is a simple **Bank Management System** developed using Java and JDBC.  
It allows users to perform basic banking operations such as:

- Account Validation  
- Check Account Balance  
- Transfer Funds Between Accounts  

 Technologies Used

- **Language:** Java  
- **Database:** Oracle Database  
- **Connectivity:** JDBC  
- **IDE:** Eclipse  
- **JDK Version:** Java SE 22
